## Title

Nádas Tavon (It's a Hungarian poem's title)

## Technologies

- Unity (Game Engine)
- Visual Studio Code
- Git & GitHub
- Other tools

## Project Purpose

The project was created by a 4-member team, and its purpose is to demonstrate
the poem named "Nádas tavon." This poem describes a peaceful place and implies that
the lyrical self had a difficult time before this peace.
The game has a simple logic: you have to jump over
obstacles that represent the difficulties of life. If the player successfully gets through
the obstacles, a short video starts that represents the peace of the poem.

## Features

- Basic gameplay mechanics
- UI system (Main Menu: Start, Difficulty, Quit, Back buttons. Resume and Restart Menu (in-game): Restart, Quit to Main, Resume buttons)
- Character movement (Jumping)
- Collision system
- Sound effects, music, and animations
- Video at the end

## Controls

- Spacebar = Jump
- Esc key = Pause

## Short Story

This project has a funny story. It was a Friday morning when my Hungarian literature teacher walked into my class
and said: "Split into 4-member teams. I will give you three lyric poems, and you have to make a presentation. 
You can create anything, just make something fabulous by Monday."
So, I went to my teammates. At first, we didn't know what we would do. As a joke, I said that I would make a game. 
I think my tone didn't carry enough sarcasm, because my teammates took it seriously. I created the project design, and then the work began.
We had only three days. It felt like a hackathon, and I was the only one who could use the Unity Engine and write code.
The girl who was the 2D artist finished the sprites later that day, so I could create the prototype. At the beginning, nothing worked.
On Saturday, I worked approximately six to eight hours to fix every single bug and implement the basic UI and game mechanics.
On Sunday, I worked five to six hours to implement the audio, sound effects, animations, and the short clip at the end.
